Understanding Media Query

  • Allow us to control,
  • the set of css rules to be applied
  • over different set of conditions.

    • Two.

    • Media type : It's a type of device on which our code will run.
      • Screen
      • Print
      • Speech
      • All
    • Media Property
      • Width
      • Height
      • Orientation
      • Screen resolution

    @media < media-type > and (media-property-condition){ < css you want > }

    Eg. @media <type of media> and <condition>{ #div1{ } #div2{ } }

    • // Small devices (landscape phones, 576px and up) @media (min-width: 576px) { ... }
    • // Medium devices (tablets, 768px and up) @media (min-width: 768px) { ... }
    • // Large devices (desktops, 992px and up) @media (min-width: 992px) { ... }
    • // X-Large devices (large desktops, 1200px and up) @media (min-width: 1200px) { ... }
    • // XX-Large devices (larger desktops, 1400px and up) @media (min-width: 1400px) { ... }
